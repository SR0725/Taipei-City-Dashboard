🚗 臺北停車場儀表板

互動式圓圈篩選停車場地圖組件

🎯 功能特色

  • 點擊拖拉創建圓圈篩選範圍
  • 根據停車場大小顯示不同顏色和尺寸
  • 身障友善和孕婦友善停車場特殊標示
  • 即時顯示篩選範圍和找到的停車場數量
  • 清除篩選按鈕重置地圖視圖
  • 響應式圖例說明停車場類型

🚀 使用說明

  1. 確保已安裝所有依賴套件：npm install
  2. 啟動開發伺服器：npm run dev
  3. 在儀表板中添加新組件，使用上述配置
  4. 組件會自動載入停車場資料並顯示在地圖上
  5. 點擊拖拉創建圓圈篩選範圍，篩選特定區域的停車場
  6. 使用「清除篩選」按鈕重置地圖視圖

📍 圓圈篩選操作

  • 點擊：選擇圓圈中心點
  • 拖拉：調整圓圈半徑範圍
  • 釋放：確認篩選範圍，顯示範圍內停車場

🎨 停車場顏色說明

  • 🔴 紅色：小型停車場 (≤100車位)
  • 🟠 橙色：中型停車場 (101-500車位)
  • 🟢 綠色：大型停車場 (>500車位)
  • 🔵 藍色：身障友善停車場
  • 🟣 紫色：孕婦友善停車場
